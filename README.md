# MERN-Crud-App

A simple CRUD application that allows creating new data in the database, reading data from the database, deleting and updating data.

## Technologies Used


### Frontend

- React.js
- React Hooks
- React-router-dom 
- Axios 
- Tailwind Css

### Backend

- React.js
- React Hooks
- React-router-dom 
- Axios 
- Tailwind Css

### Database

- MongoDb
